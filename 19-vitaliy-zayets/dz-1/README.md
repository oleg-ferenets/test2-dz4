## Основні команди для роботи з Git

### **git clone url** 
``` 
Скопіювати репозиторій в локальну папку з віддаленого репозиторію 
``` 
### **git init**
```
*Ініціалізувати новий репозиторій*
``` 
### **git status** 
```
Перегляд, що знаходиться в робочій директорії
```
### **git add <ім'я/файла>**
```
Додати попередні зміни в staging area 
```
### **git checkout** 
```
Перейти на гілку, яка є в репозиторії 
```
### **git checkout -b <ім'я/гілки> = git branch <ім'я/гілки>**
```
Створити гілку
``` 
### **git push** 
```
Залити на сервер
``` 
### **git fetch**
```
Показує що в директорії є зміни 
```
### **git commit** 
```
Коміт в свій локальний репозиторій 
```
### **git pull** 
```
Отримати з сервера 
```
### **git flow**
```
Цей набір розширень git надає високорівневі операції над репозиторієм для підтримки моделі розгалуження Vincent Driessen
``` 
### **git merge** 
```
Поєднання змін із даної гілки в гілку master
```
